Semantic HTML Cheat Sheet

Element Placement

Elements can tell developers exactly what an element does or where it's placed based on its name.

Tag Name Description
<header> Header Creates basic structure of a webpage.
<nav> Navigation Creates basic structure of a webpage. The <nav> element encloses the page's navigational links and is often placed inside the <header> or the <footer>.
<main> Main Creates basic structure of a webpage. The <main> element encloses the main content of a webpage between the header and/or navigation and the footer areas.
<footer> Footer Creates basic structure of a webpage. The <footer> element contains the page's footer content at the bottom, typically below the <body> portion of the page.

Embedding Media

These are elements used to embed media files to a website.

Tag Name Description
<video> Video Tag The <video> element allows you to add videos to your website. Requires a closing tag.
<audio> Audio Tag The <audio> element allows you to add audio files to your website. Requires a closing tag.
<embed> Embed Tag The <embed> element allows you to add any type of media to your website. Does not require a closing tag.

<figure> and <figcaption>

Elements that are typically used by developers to group media and its description.

Tag Name Description
<figure> Figure The <figure> is used to enclose media like an image, diagram, or code snippet.
<figcaption> FigCaption The <figcaption> element is used to describe media enclosed in the <figure> element.

<section> and <article>

More often than not, developers use a <section> element to define a theme for a webpage and use the <article> element to write content specifically for that theme.

Tag Name Description
<section> Section The <section> element defines elements in a document like chapters, headings, or another area of the document with the same theme.
<article> Article The <article> element holds content that makes sense on its own like articles, blogs, and comments.

Aside Element

The <aside> element contains information that is related to the main content, but not required to understand the dominant information.

Tag Name Description
<aside> Aside The <aside> element is used to mark additional information that can enhance the overall piece. Typically found in a sidebar or another location where it doesn't obstruct the main information that is being presented.
